# 高效使用Win10

## 常用快捷键

快捷键|功能
-----|----
`Win + I` |设置  
`Win + E` |文件资源管理器  
`Win + A` |操作中心  
`Win + S` |搜索  
`Win + V` |剪贴板  
`Win + Space` |切换输入法  
`Win + D` |显示桌面  
`Win + L` |锁定桌面  
`Win + Shift + S` |Windows截图  
`Win + ;` |Emoji表情  
`Win + ←` |窗口分屏在左边  
`Win + →` |窗口分屏在右边  
`Win + ↑` |窗口全屏显示  
`Win + ↓` |窗口最下化  
`Win + Ctrl + D` |创建新桌面  
`Win + Ctrl + F4` |删除桌面  
`Win + Ctrl + ←` |向左切换桌面  
`Win + Ctrl + →` |向右切换桌面  
`Win + Tab` |时间轴  
`Alt + Tab` |任务切换  
`Ctrl + Tab` |页面切换
